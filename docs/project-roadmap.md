# Project Roadmap

## Complete

- Cinematic Home hero with deterministic manual looping.
- Studio, About, Journal, Reach Us, and branded 404 routes.
- Shared navigation, footer, route titles, focus and scroll behavior.
- Mobile modal navigation with focus containment.
- Honest validated email-draft contact flow.
- Reduced-motion poster and bandwidth protection.
- Vercel and Netlify SPA fallbacks.
- Production deployment on Vercel with direct-route and browser verification.
- Automated tests, coverage gates, browser QA, code review, and project documentation.

## Before Public Launch

- Confirm the public contact mailbox.
- Validate the MP4 CDN bandwidth/cost policy.
- Decide whether Google Fonts should be self-hosted.

## Optional Next Iterations

- Add real, approved case-study content and media.
- Introduce individual Journal article routes or a CMS only when publishing workflow requires it.
- Add a consent-aware analytics solution only after metrics and privacy requirements are defined.
- Connect a backend contact service only when delivery, spam prevention, retention, and privacy rules are agreed.
- Add performance budgets and deployed Lighthouse monitoring.

## Guardrails

- Do not present conceptual Studio work as client work.
- Do not turn the Home route into a long marketing page without a new design decision.
- Do not add persistence or third-party tracking silently.
- Preserve the manual video fade/restart contract and reduced-motion no-MP4 behavior.
